{% blocktrans %} The Raspberry Pi as different modes to run in. The mode defines how the pins are numbered. If set to Board all pins are numbered with there corresponding physical location from 1 to 40. If set to BCM pins are identified by their Broadcom pin number. This page provides a nice overview of both numbering modes. If you are unsure its probably easiest to board mode as you can count the physical pins to get correct numbers. {% endblocktrans %}
